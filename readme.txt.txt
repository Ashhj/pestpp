This directory contains the source code for PEST++ along with the source code for the following
libraries which it depends on:
  1) Genie
  2) LAPACK++
  3) PROPACK
  4) several FORTRAN subroutine from the original PEST program
  
The file ".\pest++\pest++.sln" is a Visual Studio Solution.  It contains projects for PEST++
and all the the dependies listed above.  The Microsoft Visual Studio 2010
and Intel(R) Visual Fortran Composer XE 2011 Update 9  compilers were used for this effort
